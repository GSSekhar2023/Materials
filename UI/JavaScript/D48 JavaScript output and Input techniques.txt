                                        innerText
- It can display message into any container element.
    <div> <span> <p> <dd> <dt> <td> <h2> ...
- It will not support any markup or formatted content.

Syntax:
        element.innerText = "message"

                                        innerHTML
- It is similar to innerText but allows formats and markup.

Syntax:
        element.innerHTML = "message | markup";

                                        outerHTML
- It is used to remove the reffered element and add the specified element.
- InnerHTML will add specified element as child of reffered element.

Syntax:
        element.outerHTML = "<newElement> </newElement>";

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <script type="text/javascript">
        function DeleteClick(){
            flag = confirm("Delete Record\nAre you sure want to delete?");
            if(flag==true){
                document.querySelector("p").outerHTML = "<h2>Record Deleted Successfully..</h2>";      
            } else {
                alert("You canceled..");
            }
        }
    </script>
</head>
<body>
    <button onclick="DeleteClick()">Delete</button>
    <p></p>
</body>
</html>

                                        textContent
- It is similar to innerText but mostly used while adding content dynamically into element.
- If you are creating an element dynamically, then the content can be added as "textContent".

Syntax:
        element.textContent = "message";


                                         console methods

- Console is a command line tool provided by browser.
- Developers can test their commands using console.
- Console methods are contextual
        error()
        info()
        warn()
        log()
        debug()

Note: console methods must be removed from page while publishing the page.
          It is only for development not for production.

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <script type="text/javascript">
        function DeleteClick(){
            console.warn("Delete Clicked");
            flag = confirm("Delete Record\nAre you sure want to delete?");
            if(flag==true){
                console.error("OK Clicked and Record will be deleted");
                document.querySelector("p").innerText = "Record Deleted Successfully..";      
            } else {
                alert("You canceled..");
                console.info("Cancel Clicked");
            }
        }
    </script>
</head>
<body>
    <button onclick="DeleteClick()">Delete</button>
    <p></p>
</body>
</html>

                            JavaScript Input Techniques
- Input is the process of reading value from user.
- User can input a value into your application.
- JavaScript input techniques
        a) prompt()
        b) form input elements
        c) query string

                                    prompt()

- It provides an input box, which pops up in browser window.
- You can input a value, but it is just plain text content.

Syntax:
            prompt("message", "default_value");
            prompt("message");

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Input</title>
    <script type="text/javascript">
        function AddClick(){
           folderName =  prompt("Enter Folder Name","New_Folder");
           if(folderName=="") {
            alert("Folder Name Can't be Empty");
           } else if(folderName==null) {
             alert("Please provide folder name");
           } else {
              document.querySelector("p").innerHTML += "Folder Created  :" + folderName + "<br>";
           }
        }
    </script>
</head>
<body>
    <button onclick="AddClick()">Add new folder</button>
    <p></p>
</body>
</html>
