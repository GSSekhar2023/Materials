                                    Header Element
- It defines the content to display at top margin of page.
- Usually header contains
        a) Brand Name
        b) Brand Logo
        c) Search Bar
        d) Shortcut icons
        e) Navbar etc...

            path                 ./            Access files from current folder.
                                ../            Access files from previous level [one level back]
                                ../../


            display:flex                        It will keep the content side by side.
            justify-content:space-between    It will set space between the divisions.
            background-color
            color
            font-family
            font-size
            font-weight
            margin
            padding
