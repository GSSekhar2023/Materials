HTML Page Structure
1. Extentions
        .html
        .htm
2. Document Declaration
        <!DOCTYPE html>

3. Document Scope
        <html> </html>
    - It specifies the start and end of a document in browser.
    - Every HTML document scope must define the content language type.

        <html  lang="en-in">
        </html>

    - Browser Engines use Lexical scope technique. It is bottom to top approach.
    - Developers can write code outside HTML scope, which can perform better but have
     abnormal behaviour.

4. Every document comprises of 2 sections
   
            a) Head    <head> </head>
            b) Body    <body> </body>

    Head section comprises of content, which is intended to load into browser memory
     and then deliverd to browser.

    Body section comprises of content which is intended to load into browser directly.

5. Head Section of HTML Page
    - HTML head section typically comprises of following elements
            a) title
            b) link
            c) meta
            d) style
            e) script

                                             title
       - It contains information, which is displayed in the title bar of browser.
       - It is used by title bar and favorites.
       - It is also use in SEO. [Search Engine Optimization]

       Syntax:
            <!DOCTYPE html>
            <html lang="en-in">
            <head>
                <title> Flipkart | Online Shopping </title>
            </head>
            </html>

                                         Link
     - It is used to link any external document to your webpage.
       - It links
            a) Favicons
            b) Stylesheets

Ex: Creating a Favicon for our website

1. Go to your project "public/images" folder

2. Add a new file
            "favicon.ico"

3. Right click on file => Reveal in Explorer

4. Right click on icon file => open with "MsPaint"

5. Ctrl + E [ File => Properties] of MsPaint and set size  "32 x 32 pixels".

6. Draw your icons and Save [Ctrl + S]

7. Go to your Index.html page and link

    <head>
        <link  rel="shortcut icon"   href="public/images/favicon.ico">
    </head>


                                        Meta
       - Meta refers to "meta data".
     - Meta data for website is required to make the page SEO friendly and Responsive.

 Syntax: Responsive

            <meta  name="viewport"  content="width=device-width, initial-scale=1">
