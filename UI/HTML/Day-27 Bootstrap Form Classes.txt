Bootstrap Form Classes
- Bootstrap Icons
        >npm install bootstrap-icons  --save
- Bootstrap
        >npm  install bootstrap --save
- Bootstrap CSS library
        node_modules/bootstrap/dist/css/bootstrap.css

Bootstrap Margin Classes

.m            all directions
.ms        left [start]
.me        right [end]
.mt            top
.mb        bottom

value:  1 to 4

Syntax:
            <div class="me-4">

Bootstrap Padding Classes

.p            all directions
.ps            left
.pe            right
.pt            top
.pb            bottom

Value : 1 to 4

Syntax:
            <div class="pe-4">

Bootstrap Border Classes

.border
.border-{size}  1 to 3
.border-{contextual}  primary, secondary, success, danger, warning, info, dark, light..

Syntax:
            <div  class="border border-3 border-primary">

.rounded
.rounded-{size} 1 to 3
.rounded-pill
.rounded-circle

Syntax:
            <img  class="rounded  rounded-circle">

Bootstrap Button Classes

.btn
.btn-{contextual}  primary, danger, warning, success...
.btn-{lg | sm}  size
.btn-outline-{contextual}
.btn-group
.btn-group-vertical
.btn-close
.btn-link
.btn-toolbar

Syntax:
    <button  class="btn btn-outline-primary btn-sm"> Insert </button>

    <div class="btn-group">
        <button></button>
        <button></button>
    </div>

Bootstrap Form Element Classes

.form-control            : text, password, number, date, email, url, textarea
.form-select            : dropdown and listbox
.form-range            : range
.form-color            : color
.form-file                : file
.form-label            : labels
.form-check-input    : radio and checkbox
.form-check-label    : labels used for checkbox and radio
.form-switch            : radio and checkbox container

Bootstrap Width class
.w-{25, 50, 75, 100}
