                                        Main Element
- Main in computer programming is considered as "Entry Point".
- Usually the Getstarted content is kept in main element.

Controlling Background Image using CSS:
   
background-size                    : width, height, contain, cover
background-repeat                : no-repeat, repeat, repeat-x, repeat-y
background-position            : sets position on screen [x, y]
                                        [top, center, bottom]  vertical
                                        [left, center, right]
                                     
background-attachment        : scroll, fixed
background-color                : any color
background-image                : url()  any image path.

CSS Colors provide:
    rgb()                    rgb(redValue, greenValue, buleValue)   0 to 255
    rgba()                    rgba(red, green, blue, alpha)                 apha 0 to 1

To Keep any content center screen:
1. Keep all elements in one container

            <body>
            <main>
                ....
                ....
            </main>
            </body>

2. Apply effects for body
            {
             display:flex;
             justify-content:center;
             align-items:center;
            }
