                                        Gradient Colors
- It is a combination of multiple colors.
- You have to set gradient only by using "background-image".

        a) Linear Gradient
        b) Radial Gradient

- Linear-Gradient : It sets horizontal and vertical orientation for colors.

Syntax:
            background-image: linear-gradient(direction, colors);
            background-image: linear-gradient(to right, red, green);

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        body{
            height: 700px;
            background-image: linear-gradient(to top left, pink 20% ,yellow,green);
        }
    </style>
</head>
<body>
    <div class="container">
        container text 
        <h2>Heading Text</h2>
    </div>
</body>
</html>


- Radial-Gradient:  It sets orientation from center.

Syntax:
            background-image: radial-gradient(colors);

Ex:
     body {
            height: 500px;
            background-image: radial-gradient(red 20%, yellow, green);
        }

<!DOCTYPE html>

                                       CSS Units
- Units are required to set dimensions for any element.
- CSS units are classified into 2 types

        a) Absolute Units
        b) Relative Units

Absolute Units are individual and not effected by any adjacent or parent element.

                cm            centimeters
                mm
                Q
                in
                pt
                pc
                px

Relative units depends on the parent element or root element.

                em                element size    
                rem            root element size

Syntax:    
            parent {
                font-size : 40px;
            }
            child-1 {
                font-size: 1em;
            }
            child-2 {
                font-size: 1rem;
            }
EX:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Units</title>
    <style>
        .container{
            font-size: 40px;
            border: 1px solid red;
            width: 500px;
            color: blue;
        }
        p{
            font-size: 0.5em;
            color: red;
        }
        span{
            font-size: 1.5rem;
        }
    </style>
</head>
<body>
    general text 
    <div class="container">
        container text 
        <p>paragraph text</p>
        <span>span text </span>
    </div>
</body>
</html>

                                        CSS Margins
margin            : short hand for all directions
                     
Syntax:
     margin : 20px;                            all directions 20px;
     margin : top, right, bottom, left           all directions in short hand
   
Ex:
     margin: 0px, 30px, 0px, 30px;                    

margin-left            : only left margin
margin-right            : only right margin
margin-top            : only top margin
margin-bottom        : only bottom margin

                                        CSS Padding
- It is distance between content and border.

Attributes:
padding                    : short hand all directions
padding-left
padding-right
padding-top
padding-bottom

                                        CSS Border
Attributes:

border                    : short hand for all border effects
border-width            : sets only size
border-style            : sets line style, solid, double, dotted, dashed, groove etc..
border-color            : sets color

Border Orientation

border-left            : short hand for left border
border-left-width
border-left-style
border-left-color

.... similarly all directions are defined....
