                                    How to Embed Images
<img>

Image Attributes

1. src                : It specifies the name and path of image.

                     <img  src="images/realme.jpg">

2. alt                : It defines the alternative text to display when image fails to load.

                      <img  alt="text to display"  src="images/pic.jpg">

3. title                : It defines the tooltip to display when mouse is over image.

                       <img  alt="text"  title="some tips" src="images/pic.jpg">

4. border            : It sets border for image.


                        <img  border="2">

5. width & height    :  It defines width and height of image in pixels or in "%".
                       Size in "%" is used for fluid images.

                        <img  src="images/pic.jpg"  width="50%"  height="300">


6. align            : It can align image only left or right, there is no center align for image.
                      It is required to set text wrapping.


                        <img src="images/pic.jpg" align="left | right">

7. hspace & vspace :   Sets horizontal and vertical space between image and text.

                        <img src="images/pic.jpg"  hspace="15" vspace="20">


Advanced Attributes:

1. crossorigin
2. decode
3. srcset
4. importance
