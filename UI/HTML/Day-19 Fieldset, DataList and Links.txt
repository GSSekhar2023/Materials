Datalist with Terms and Definitions

- Data List is defined by using
        <dl>
- Data list contains terms defined using
        <dt>
- Every terms have definition defined using
        <dd>

Syntax:
        <dl>
            <dt> Term </dt>
            <dd> Definition </dd>
        </dl>

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <dt>Styles</dt>
        <dd>It defines styles for HTML.</dd>
        <dt>JavaScript</dt>
        <dd>It manipulates DOM.</dd>
    </dl>
</body>
</html>

                                Fieldset and Legend

- Fieldset sets a frame for your content.
- Legend sets a caption for frame.

Syntax:
        <fieldset>
            <legend> Title </legend>
             .........................
             your content
             ..............
        </fieldset>

                                    Links in HTML

- Links are used to design navigation for website.
- Link is clickable text, picture or graphic, that navigates the user to any specified location.
- Links are classified into 2 types
        a) Intra Document Links
        b) Inter Document Links

- Intra Document links navigate user from one location to another with in the same page.
- Inter document links will navigate user from one page to another.

- Links are created by using HTML anchor element  <a>.

Intra Document Links:
1. Define ID for any element
        <h2 id="home">

2. Refer ID by using <a> anchor

        <a href="#home> Home </a>

Ex:
