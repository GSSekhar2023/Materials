                                     CheckBox
- It allows the user to check and  uncheck any option.
- User can choose one or multiple from a group of choices.

Syntax:
        <input type="checkbox"  checked   value=""> <label> Title </label>


Ex:

                                    Dropdown List

- It can configure a list of items, from where user can select any one item from list.
- It is easy to add or remove items dynamically with script.
- Dropdown List is created by using
        <select>
- Items in list are created by using
        <option>
- Items can be grouped by using
        <optgroup>

Syntax:
        <select>
            <optgroup label="Title">
               <option> Item-1 </option>
               <option> Item-2 </option>
            </optgroup>
        </select>

- <select> element can have attributes:   id, name, class
- <option> element can have attributes
   
        selected            It keeps the item selected automatically.
        disabled            It will not allow to select item
        value                It defines the value to submit.

Syntax:
        <option  selected>
        <option  disabled>
        <option  value="">

Note: <select> and <option> are RC data type. They will not allow any icon or format
           for element.

                                     ListBox
- It is same like dropdown list, but allows to select multiple.
- All elements are same as dropdown.
- Dropdown will transform into listbox by using 2 attributes
        a) size
        b) multiple

Syntax:
            <select size="3"  multiple>
               <optgroup label="Title">
                <option value="" selected  disabled> </option>
               </optgroup>
            </select>

                                        Buttons
- Buttons are used for confirming user actions.
- It includes various actions

        Navigation            : previous, next, first, last, goto
        Record                : New, Register, submit, reset, insert, update, delete
        Miscelaneous        : print, save, open, close, minimize etc..

- HTML provides 2 types of buttons
        a) Generic
        b) Non-Generic

- Generic buttons have pre-defined functionality
        a) submit            : it can submit form data
        b) reset            : it can reset form data

    HTML 4 Buttons                    : RC data type
        <input type="submit">
        <input type="reset">

    HTML 5 Buttons
         <button type="submit"> text | images | icon  </button>
         <button type="reset"> text | image | icons </button>

- Non-Generic button is ordinary button without functionality.

    HTML 4
        <input type="button" value="">

    HTML 5
        <button type="button"> </button>
