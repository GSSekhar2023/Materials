CSS Display

- display attribute comprises of values
        1. none
        2. inline
        3. block
        4. flex
        5. grid

                                        display:none
- It hides HTML elements and removes the space allocated for the element.

FAQ: In CSS we can hide any element by using 2 techniques
        a) display:none
        b) visibility:hidden
        what is difference between none and hidden?
Ans: display:none,  hides and removes the space allocated.
       visibility:hidden,  hides without removing the allocated space.

                                display:inline, display:block
- Inline is used to keep element side-by-side.
- block is used to keep element one below another.
- Inline is only for element, where as flex is for container.

Ex:

FAQ: What is difference between "inline" and "flex"?
Ans : Inline is used for inidividual elements.
        Flex is used for container that contains elements.

Syntax:
        ul {
            display:inline;            // not valid
         }
        li {
            display:inline;        // valid
          }
        ul {
            display:flex;
        }

        Inline can't adjust space between or around the content.
        Flex provides various options to control the alignment of elements in container.

                                display:flex & flex attributes
- flex is for container and flex attributes are used to control elements in container

    a) flex-wrap            : wrap, no-wrap
    b) flex-direction        : row, row-reverse, column, column-reverse

- flex is not for displaying side-by-side, it can handle various directions.
- It is flexible display horizontally and vertically.
- Flex box can make the elements fit according to width and height.


                                Flex Direction
                                -------------------
- It controls the orientation of elements in container.
- flex-direction : row   is default, if you want, you can change to column.
- Flex alignments are controlled by using the attribute
        a) justify-content     : space-between, space-around, space-evenly
        b) align-items     : baseline, center [vertical - column] [horizontal - row], end

     - Flex:wrap is used to control the wrapping of elements in container.
     - wrapping is the process of displaying content into multiple rows and columns when
      width for content is not sufficient.

Syntax:
        .container {
            display:flex;
            flex-wrap: no-wrap | wrap;
        }

                                    Display-Grid
- It is used to create responsive layout, which can change the number of rows and columns dynamically.
- It also allow to change the position of content in any row or column.

Syntax:
        {
         display:grid;
         grid-template-columns: fractions;
         grid-row: number;
         grid-column:number;
        }
