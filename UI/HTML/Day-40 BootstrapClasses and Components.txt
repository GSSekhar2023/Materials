

Bootstrap Classes and Attributes

- Bootstrap provides type selectors, which apply styles for elements automatically when linked to HTML page.
- Bootstrap provides pre-defined classes and attributes to control elements.
   
        classes            : comprises of styles
        attributes            : comprises of logic

Bootstrap Container classes:
- container is used to keep content with margin and padding.
- bootstrap have default containers for various devices

        .container                mobile devices
        .container-sm        tablet [tabs]
        .container-lg            desktop
        .container-xl            laptop upto 1200px
        .container-xxl            laptop upto 1600px
        .container-fluid        responsive

Bootstrap Box Model Classes:

Margins:
    .m            all directions
    .ms        left [start]
    .me        right [end]
    .mt            top
    .mb        bottom

    break-point
    .m-{ }        => It is size for margin which can be 1 to 4
    .m-4        
    .mt-4
    .ms-3

Syntax:
        <div class="mt-3 mb-3 ms-2 me-2"> Text </div>
        <div class="m-4"> Text </div>

Padding:
    .p            all directions
    .ps
    .pe
    .pt
    .pb

    break-point
    .p-{size}        from 1 to 4

Border:
    .border
    .border-{size}                    from 1 to 3
    .border-{contextual}            primary, danger, warning, info, success, dark, light ..

Border Radius:
    .rounded
    .rounded-{size}                from 1 to 3
    .rounded-circle
    .rounded-pill

Width and Height:
    .w
    .w-{}                            from 25, 50,75, 100
    .h
    .h-{}                            from 0, 25, 50,100
