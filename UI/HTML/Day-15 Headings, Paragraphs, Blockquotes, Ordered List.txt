                                     Headings in HTML
                                    -------------------------

- HTML headings are defined by using
    <hn>

- 'n' refers to level number of heading 1 to 6.

- Headings are used to make your topic SEO friendly.

Syntax:
            <h1> Text </h1>
            ............
            ............
            <h6> Text </h6>

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
     <h1>Web Technologies</h1>
     <h2>Front-End</h2>
     <h3>HTML</h3>
     It is a markup language.
     <h3>CSS</h3>
     It defines styles.
     <br><br>
     <font size="4"><b>JAVASCRIPT</b></font>
     <br><br>
     It is a language for DOM.
     <h2>Back-End</h2>
     <h3>MongoDB </h3>
     It is a database.
     <h3>Node JS</h3>
     It is server side programming.
</body>
</html>

FAQ: Why to define text in heading style, if we can configure same appearence without heading style?
Ans: To keep our topics in page SEO friendly. So that search engines can easily summarize your topics in page.

Note:
      - Never use too many headings in a page. It may lead to SPAM.
      - Never use headings to highlight any word or sentence.

FAQ: How to remove the pre-defined style given for heading?
Ans: By using CSS inheritance attribute "unset".

Syntax:
        <style>
            h1 {
                all:unset;
            }
        </style>

FAQ: Can we change the appearence of pre-defined heading styles?
Ans:  Yes. By using style attributes.

Syntax:
        <style>
            h1 {
                font-family: Algerian;
                font-size:35px;
             }
        </style>


                            Paragraphs and Blockquotes
                            ---------------------------------------
- Paragraph is defined by using the element <p>
- Blockquote is defined by using the element <blockquote>

Syntax:
        <blockquote>  your text </blockquote>

FAQ:
1. How to configure first line indent for paragraph or blockquote?
A. By using CSS attribute "text-indent".

Syntax:
        p, blockquote {
            text-indent: 50px;
        }

2. How to design a dropcap?
A. You can access first letter of paragraph by using CSS class "::first-letter".
     You can set as dropcap using : font, style, size and float.

          float:left;

                            Ordered and Unordered List
- Ordered list is defined by using
        <ol>
- Ordered list is an auto numbering style for a list of items.
- Every item in list is defined with
        <li>

Syntax:
        <ol>
          <li> Item-1 </li>
          <li> Item-2 </li>
        </ol>

- You can change the numbering style by using "type" attribute
   
          type = "a, A, i, I, 1"

          <ol  type="a">

- You can define from where the numbering have to start by using "start" attribute

          start = "5"        => level number

- You can also reverse the numbering, new from HTML 5.

          "reversed"

Syntax:
          <ol  reversed>

Ex:
    <ol type="a" reversed start="2">
        <li>HTML</li>
        <li>CSS</li>
        <li>Bootstrap</li>
        <li>JavaScript</li>
     </ol>

- You can keep orderlist nested.
