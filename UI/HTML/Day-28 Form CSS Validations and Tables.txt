CSS Classes for Form Validation

:required                : It defines effects for elements which are specified with required
                          attribute.

:in-range                : It verifies the value range that you input into any element.
                          It can apply effect only if the value is in specified range.

:out-of-range            : It sets effects if value is not within the specified range.

:valid                    : It sets effects when field is valid.

:invalid                : It sets  effects when field is invalid.

:checked                : It is used to verify checked attribute of radio or checkbox

Note: valid & invalid can verify any type of validation
            required, minlength, maxlength, min, max, pattern, email, url etc..

                                   Tables in HTML

- Tables arrange content in rows and columns.
- Table allows to change data dynamically.
- It is easy for adding, updating and removing data from table.
- It is good for sorting and filtering data using table.
- Table in HTML is configured by using following elements

        <table>
        <caption>
        <thead>
        <tbody>
        <tfoot>
        <tr>
        <th>
        <td>
        <colgroup>

Table Attributes:
1. Frame, Rules and Border

    Frame            : It is defined for entire table. It have options
                        "box, void, rhs, lhs, above, below"

                     <table  frame="box|void|rhs...">
   
    Rules            :  Rules are defined for rows, cols, groups and all.

                      <table  rules="all | rows | cols | groups | none">

    Border        : It is for cells in table. It uses  1=true 0=false.

                       <table border="1"  frame="void">


Note: You can't set border if "rules" attribute is defined in table. You have to remove rules to set border.

2. cellspacing and cellpadding

        cellspacing            : It sets space between cells.
        cellpadding            : It sets space between cellborder and its content.

            <table  cellspacing="10" cellpadding="10">
