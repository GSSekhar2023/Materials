                                        aside

 - It represents content which is not relative to current page.
 - It is mostly used for ads.
 - It is also used for navigating to the networks of current page. [Social Bar]

CSS Styles
a) position:absolute        : removes for specified location, but keeps along with content
b) position:fixed            : removes from specified location, and keeps the position fixed.
                              [not with content, it is with page].

                                        dialog

- It is used to create popup window in page.
- Dialog is by default hidden, you can show by using "open" attribute.

Syntax:
        <dialog open>
        ...... your ..content...
        </dialog>

                                    Figure with Caption
- <figure> is an element used to encapsulate a picture with caption so that it can be SEO friendly.

Syntax:
        <figure>
            .... your image....
            <figcaption> Caption </figcaption>
        </figure>

Ex:

                                    Nav with Menu
- The navigation is defined in page by using <nav> element.
- If naviagation comprises of various categories then better use <menu>

Syntax:
    <nav>
        <menu>

        </menu>
    </nav>

                                      Details and Summary
- Details and summary is used to create expandable and collapsable content in page.

Syntax:
        <details>
            <summary> Title </summary>
            .......
            .......
        </details>

- Details is by default in collapsed state, you can use "open" attribute to expand.

        <details open>

FAQ: What is ":hover"?
Ans : It is a CSS class that defines effects when mouse pointer is over the element.

FAQ: What is ":active"?
Ans: It is a CSS class that defines effects when you hold down mouse button on element.
