FAQ: What are the issues that you face with CSS as developer?
Ans:
- Code Resuability is not good in CSS
- Code Extensibility is not easy
- Accessibility of elements with selectors is not enough at high level.


                            CSS Pre-Processors [Compilers]

- Pre processors provide lot of programming features to developer for writing CSS.
- It is a different language which is not understandable to browser.
- It is transcompiled into CSS.
- The pre-processors are
            a) Sass
            b) Less

                                        Sass
                        [Syntactically Awesome Style Sheet]
- It is a css pre-processor.
- It provides programming features to make css simplified.
- It enables reusability and easy extensibility.
- It is trans-compiled into CSS.

Install Sass on your PC:

    C:\> npm install -g  sass

- Create a new folder in your project by name "scss"
- Add a new file with extention ".scss"

                effects.scss
.container {
  border:2px solid red;
  padding: 20px;
  background-color: yellow;
  h2 {
    color:red;
  }
  p {
    color:gray;
    font-style: italic;
  }
}

- Open the sass file location in Terminal

        D:\flipkart\scss> sass  effects.scss   effects.css


- Link CSS file to webpage

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="../scss/effects.css">
</head>
<body>
    <div class="container">
        <h2>Sass</h2>
        <p>Syntactically Awesome Stylesheet</p>
    </div>
</body>
</html>
